Утилита sort_study предназначена для исследования сортировок на эффективность.

В текущей версии утилита позволяет генерировать псевдослучайные данные для
тестирования на них и измерять время работы на сгенерированных тестах.

Генерация:
Генерация производится командой "generate". После нее необходимо указать
максимальное количество элементов в массиве для теста и путь к папке, в
которой требуется сохранить тесты.
ОСТОРОЖНО! ВСЕ ФАЙЛЫ В УКАЗАННОЙ ПАПКЕ ПРИ ГЕНЕРАЦИИ БУДУТ УДАЛЕНЫ!!!
Если последний элемент в пути не существовал, то он будет создан автоматически.
Тесты в папке сохраняются в отельные файлы. Число в названии файла указывает
на количество элементов в тесте.

Запуск тестов:
Тестирование производится командой "run". После нее необъодимо указать тот же
путь, что и при генерации. В текущей версии тестирование производится на всех
реализованных в утилите сортировках и на стандартной сортировке list.sort.
Результаты тестирования сохраняются в подпапку sorttest.results указанной
папки. Для каждой сортировки результаты сохраняются в отдельном файле.
Имя файла содержит название соответствующей сортировки.

Структура файла с результатами:
Каждая строка файла описывает результаты тестирование сортировки на одном
тесте. Строка состоит из одного целого числа и двух чисел с плавающей точкой.
Первое обозначает размер теста. Следующие два - границы доверительного
интервала. Доверительный интервал строится минимум по пяти, максимум по
пятидесяти измерениям. Доверительная вероятность равна 95%.

Реализованные сортировки:
* сортировка пузырьком
* сортировка вставками
* сортировка кучей
* сортировка Шелла
* сортировка Хоара
* сортировка слиянием
